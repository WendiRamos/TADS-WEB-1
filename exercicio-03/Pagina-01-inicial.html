1- Cite e explique as formas de usar um código CSS.

R:Existem três formas para incluir o código CSS em seu projeto

  1. Inline
  2. Interno
  3. Externo

Inline: A primeira forma de aplicar CSS a uma página é utilizando o atributo style em elementos do HTML.

Interno: A segunda forma é utilizar a tag style dentro do head da página HTML.

Externo: E a última maneira de aplicar CSS é criar um ou mais arquivos com extensão .css e incluí-los na estrutura head do HTML.
